# IC10 Resource Guides

Curated external resources for IC10 programming in Stationeers.

These guides are generated when you ask for help with IC10 automation. They contain links to external resources rather than copied code.

## How It Works

When you ask for IC10 examples or scripts:
1. External sources are searched (GitHub, Wiki, Reddit, forums)
2. Results are evaluated and ranked by quality/relevance
3. A guide is created with links, summaries, and code previews
4. Full code remains at the original source

## Sources Searched

- **GitHub**: 9 community repositories with IC10 scripts
- **Stationeers Wiki**: stationeers-wiki.com
- **Reddit**: r/stationeers discussions
- **Web**: Tutorials, forum posts, guides

## Guide Format

Each guide contains:
- **Summary**: What was found
- **Best Resources**: Top matches with code previews (10-15 lines)
- **Tutorials & Guides**: Learning resources
- **Community Discussions**: Relevant forum threads
